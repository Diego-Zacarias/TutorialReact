# Breakpoints

Breakpoints for responsive layout:

<table>
  <tr>
    <th>Extra Small</th>
    <td>Default</td>
  </tr>
  <tr>
    <th>Small</th>
    <td>{BreakpointSize.sm} px</td>
  </tr>
  <tr>
    <th>Medium</th>
    <td>{BreakpointSize.md} px</td>
  </tr>
  <tr>
    <th>Large</th>
    <td>{BreakpointSize.lg} px</td>
  </tr>
  <tr>
    <th>Extra Large</th>
    <td>{BreakpointSize.xl} px</td>
  </tr>
</table>

Using styled components:

```js
import { BreakpointSize, breakAt } from "Breakpoints";

const Title = styled.h1`
  font-size: 1rem;

  ${breakAt(BreakpointSize.md)} {
    font-size: 2rem;
  }
`;
```
